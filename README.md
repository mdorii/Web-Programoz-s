- react.html – React CRUD (useState)
- spa.html – SPA (Kereső + Ár Kalkulátor)
- fetchapi.html – Fetch API + Flask backend
- axios.html – React + Axios + Flask backend
- oojs.html – OOP JS Torta Generátor (class, extends, super, appendChild)
